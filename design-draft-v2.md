# Bounty Command Center — Visual Draft v2

## Direction
Future mech armor + battle-worn hunter guild + cultivation energy system.

Reference mood distilled from the user's requested direction:
- massive futuristic armor surfaces
- dense but readable tactical HUD
- cold metal with energy veins
- strong rank / advancement / power-growth symbolism
- asymmetrical panel silhouettes rather than perfectly aligned rectangles

## Core Visual Principles
1. **Armor first**: panels feel like mech plating, not SaaS cards.
2. **Battle damage is controlled**: scratches, scorch marks, patched seams, serial codes.
3. **Cultivation energy is internal**: glowing cores, ring gauges, meridian-like routes.
4. **Guild hierarchy is visible**: seals, ranks, bounty grades, authority stamps.
5. **Asymmetry is intentional**: outlines and corner cuts should not all mirror each other.

## Surface Language
- hard beveled edges
- offset panel seams
- cut corners, broken corner brackets
- embedded code markings and caution labels
- layered glass over armor base
- faint energy scanlines and sector rings

## Layout Rule
Top / center / side modules should feel mounted onto a mech chassis. Avoid clean perfect symmetry. Leave some modules slightly offset vertically or inset with different corner treatments.

## Key Components
### 1. Guild Armor Header
- guild insignia
- command status
- tactical weather/time strip
- warning lights / authorization stamp

### 2. Core Reactor Hero
- central energy nucleus
- cultivation stage / combat level
- ring gauges for spirit pressure, armor integrity, mission load
- floating micro labels and segmented armor frame

### 3. Prime Bounty Plate
- looks like a reinforced wanted contract
- metallic strips, issue number, hazard grade, scorch marks
- reward area styled like locked reward canisters

### 4. Hunter Dossiers
- not employee cards; armored hunter dossiers
- rank crest, weapon path, advancement progress, armor condition
- each card can have different silhouette accents

### 5. Quick Actions
- heavy switch-like buttons
- some actions appear as embedded toggles, some as mechanical tabs

## Motion Guidance
- slow breathing glows only on important energy zones
- sweeping scanlines on hero and reactor areas
- no hyperactive neon everywhere
- warning cards may pulse softly

## Copy Tone
Prefer:
- 悬赏令 / Prime Bounty
- 猎人档案 / Hunter Dossier
- 灵核 / Reactor Core
- 装甲完整度 / Armor Integrity
- 突破进程 / Ascension Progress
- 战术授印 / Tactical Seal

Avoid plain office words like dashboard, task list, user management.
